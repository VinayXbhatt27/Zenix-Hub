## ğŸš€ Getting Started

### Prerequisites

- Node.js (v16 or higher recommended)
- npm or yarn

### 1. Clone the repo

```bash
git clone https://github.com/VinayXbhatt27/Zenix-Hub.git
cd Zenix-Hub
```

### 2. Install dependencies

```bash
npm install
# or
yarn install
```

### 3. Run the app

```bash
npm run dev
# or
yarn dev
```

The app will be available at `http://localhost:5173` by default.
